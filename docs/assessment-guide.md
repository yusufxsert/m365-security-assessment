# Assessment Guide / Bewertungshandbuch

## 🇩🇪 Wie wird bewertet?

| Status | Bedeutung |
|---|---|
| ✅ OK | Konfiguration entspricht Best Practices |
| ⚠️ WARN | Verbesserungspotenzial vorhanden |
| ❌ FAIL | Kritische Lücke — sofortiger Handlungsbedarf |

## 🇬🇧 Assessment Criteria

Each check maps to Microsoft Security Baseline or CIS M365 Benchmark recommendations.
Run the assessment regularly (monthly recommended) to track your security posture over time.
